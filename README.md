# Website Product Monitor
A Python application that monitors for product restocks. Currently supports BestBuy.

## Get started
You can get started by cloning this repository and editing the `bestBuySkus.txt` file with the skus of the products you would like to monitor. Next, you can start the program by typing `python bestBuy.py` in your terminal, otherwise if you have any proxies, just insert them in `proxies.txt` and then type `python bestBuy.py` in your terminal.
